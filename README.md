# My Portfolio

## Description

This is my personal portfolio project showcasing my skills and projects.

## Technologies Used

- HTML
- CSS, Sass, Bootstrap
- React + Typecript
- Icons from Font Awesome

## Features Implemented

- About Me section
- Projects section
- Contact section

- Responsive layout
